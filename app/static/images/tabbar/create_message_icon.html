消息图标生成器

使用说明：

  1. 右键点击下面的图标
  2. 选择"另存为图片"
  3. 保存为 message.png (未选中状态)
  4. 将图标颜色改为蓝色，保存为 message_.png (选中状态)

保存步骤：

  1. 右键点击选中的图标
  2. 选择"另存为图片"或"复制图片"
  3. 保存到 D:\Develop\Jxq2509_1\app\static\images\tabbar\ 目录
  4. 分别保存为 message.png 和 message_.png
  5. 确保图片尺寸为 81x81 像素
